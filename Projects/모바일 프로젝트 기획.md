# Salvio: Git 프로젝트 진행상황 모니터링 - 1

## 한 줄 설명

GitHub Repository의 커밋, 브랜치, Pull Request 상태를 모바일에서 확인할 수 있는 개발 프로젝트 모니터링 앱.

---

## 핵심 구현

- GitHub 로그인
- Repository 목록 조회
- 프로젝트 대시보드
- Pull Request 상태 확인
- 커밋 활동 통계

---

## 주요 기능

### GitHub 로그인
- GitHub OAuth 인증
- 사용자 Repository 목록 조회
- 선택한 Repository 저장

### 프로젝트 대시보드
- 최근 커밋 현황 조회
- Open / Merged PR 개수 표시
- 브랜치 수 표시
- 최근 활동 요약

### Pull Request 모니터링
- Open / Closed / Merged 상태 조회
- 리뷰 상태 확인
- 작성자 / 담당자 표시

### 브랜치 상태 확인
- 브랜치 목록 조회
- 마지막 커밋 시간 표시
- 오래된 브랜치 감지

### 작업 통계
- 작업자별 커밋 수 조회
- 기간별 커밋 활동 그래프
- 파일 변경량 통계

---

## 기술 스택

- Flutter
- GitHub REST API
- OAuth 로그인
- Dio
- Hive 또는 SQLite
- Riverpod 또는 Provider
- Chart 라이브러리

---

## 데이터 흐름

```text
GitHub OAuth 로그인
  → Repository 목록 조회
  → 선택 Repository 저장
  → 커밋 / PR / 브랜치 정보 조회
  → 앱에서 대시보드로 시각화
```

---

## 구현 포인트

- GitHub REST API 연동
- OAuth 인증 처리
- Repository / PR / Commit 데이터 가공
- 로컬 캐싱으로 최근 조회 데이터 저장
- 차트 기반 프로젝트 활동 시각화

# SoundSpot: 위치 기반 음악 방명록

## 한 줄 설명

특정 장소에 음악과 짧은 메시지를 남기고, 해당 위치에 방문한 사람들이 음악 방명록을 확인할 수 있는 위치 기반 음악 공유 앱.

---

## 핵심 구현

- 위치 기반 음악 등록
- 주변 음악 방명록 조회
- 음악 검색 및 저장
- 사용자 로그인
- 좋아요 / 즐겨찾기

---

## 주요 기능

### 위치 기반 음악 남기기
- 현재 위치에 음악 등록
- 장소명 / 메모 / 감정 태그 저장
- 공개 / 비공개 설정

### 주변 음악 조회
- 내 주변에 등록된 음악 목록 조회
- 지도 기반 음악 마커 표시
- 거리순 정렬

### 음악 검색
- 곡 제목 / 가수명 검색
- Spotify 또는 YouTube 링크 저장
- 미리듣기 링크 연결

### 음악 방명록
- 장소별 음악 목록 조회
- 짧은 메시지 작성
- 좋아요 / 즐겨찾기 등록

### 사용자 관리
- 소셜 로그인
- 내 음악 기록 조회
- 내가 좋아요한 음악 조회

---

## 기술 스택

- Flutter
- Java 17
- Spring Boot
- Spring Data JPA
- MySQL
- Kakao Map API
- Spotify API 또는 YouTube Data API
- OAuth 로그인
- Riverpod 또는 Provider

---

## 데이터 흐름

```text
사용자 위치 확인
  → 장소 선택
  → 음악 검색
  → 음악 + 메시지 저장
  → 지도에서 주변 음악 조회
```
