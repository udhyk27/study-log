# 제목 후보
# DevFlow
# GitPulse
# RepoBoard

Git 기반 프로젝트 진행 상황과 협업 상태를 모바일에서 모니터링할 수 있는 개발 협업 앱

* GitHub 로그인
→ OAuth 인증
→ Repository 목록 조회
→ 사용자 프로필 조회

* 메인 대시보드
→ 내 GitHub 활동 요약
→ 참여 중인 Repository 표시
→ 진행 중인 작업 상태 표시
→ 최근 커밋 / PR / Issue 활동 표시

* 인기 Repository 탐색
→ 기간별 Star 증가량 기준 인기 Repo 조회
→ Today / Week / Month 필터
→ 언어별 Repository 필터링
→ Star 수 / Fork 수 / Issue 수 표시

* 사용자 검색
→ GitHub 사용자 검색
→ 사용자 프로필 조회
→ Public Repository 목록 조회
→ 팔로워 / 팔로잉 수 표시
→ 주요 사용 언어 확인

* 프로젝트 진행 대시보드
→ 전체 진행률 표시
→ Open / Merged PR 통계
→ 브랜치 현황 조회
→ 최근 작업 내역 표시

* 브랜치 모니터링
→ ahead / behind 비교
→ 병합 여부 확인
→ 마지막 커밋 시간 표시
→ 오래된 브랜치 감지

* Pull Request 관리
→ Open / Closed / Merged 상태 조회
→ 리뷰 상태 확인
→ 담당자 표시

* 작업 관리
→ GitHub Issue 기반 작업 티켓 조회
→ 진행 상태 관리
→ 담당자 지정
→ 우선순위 설정
→ 마일스톤 / 라벨 기반 분류

* 협업 기능
→ 작업별 댓글
→ 개발 일정 공유
→ 프로젝트 공지 확인
→ 작업 히스토리 조회

* 프로젝트 통계
→ 작업자별 커밋 수
→ 기간별 활동량 분석
→ 파일 변경량 통계
→ 프로젝트 진행 그래프

* 데이터 시각화
→ 차트 기반 프로젝트 분석
→ 팀 활동량 표시
→ 작업 진행 흐름 표시
→ Star 증가 추이 표시
