1강 컴퓨터 통신망의 소개

분산 시스템
- 컴퓨터 자원을 분산하여 자원 공유, 확장성, 신뢰성 향상
분산 시스템 분류
- 강연결 분산시스템 → 데이터 흐름 기계, 멀티프로세서
- 약연결 분산시스템 → LAN, MAN, WAN
LAN/MAN/WAN
- LAN: 10m~1km
- MAN: 10km~1000km
- WAN: 10000km~40000km
통신
- 서로 떨어진 지점 간 정보 전달 과정
데이터 통신망
- 데이터 통신 기술 + 네트워크 기술
- 컴퓨터 통신망, 정보 통신망, 약연결 분산시스템
통신망 역사
- SAGE: 최초 컴퓨터 통신 시스템
- SABRE: 항공 예약 시스템
- ARPANET: 인터넷의 시초, TCP/IP 개발
- ALOHA: 무선 패킷 교환망
- TELNET: 상용 패킷 교환망
데이터 통신 시스템
- DTE: 단말 장치
- DCE: 신호 변환 장치
- CCE: 통신 제어 장치
프로토콜
- 통신 규약
프로토콜 요소
- 구문(Syntax): 데이터 형식
- 의미(Semantics): 제어 정보
- 타이밍(Timing): 전송 순서, 속도
네트워크 구조
- SNA(IBM)
- DNA(DEC)
- OSI
시험 단골
- 프로토콜 3요소 → 구문, 의미, 타이밍
- ARPANET → 인터넷 시초
- LAN, MAN, WAN 구분

2강 데이터 통신의 기초

변조(Modulation)
- 정보를 반송파에 실어 전송
변조 방식
- AM: 진폭 변조
- FM: 주파수 변조
- PM: 위상 변조
펄스
- 매우 짧은 전자기 신호
- 구성 요소 → 진폭, 폭, 위치
PCM(Pulse Code Modulation)
- 아날로그 → 디지털 변환
과정
- 표본화(Sampling)
- 양자화(Quantization)
- 부호화(Encoding)
나이퀴스트 정리
- 최고 주파수의 2배 이상으로 표본화해야 복원 가능
문자 코드
- Baudot: 5비트
- ASCII: 7비트, 128문자
- EBCDIC: 8비트, IBM
- BCD: 10진수 표현
- Unicode: 국제 표준 문자 코드
전송 방향
- 단방향(Simplex): 한 방향
- 반이중(Half Duplex): 양방향 교대
- 전이중(Full Duplex): 양방향 동시
전송 모드
- 병렬 전송: 근거리
- 직렬 전송: 원거리
동기 방식
- 비트 동기
- 문자 동기
전송 방식
- 동기식 전송: 블록 단위, SYN 사용
- 비동기식 전송: 문자 단위, Start/Stop 비트 사용
시험 단골
- PCM 과정
- 나이퀴스트 정리
- ASCII=7비트
- EBCDIC=8비트
- 단방향/반이중/전이중 비교

전송 매체
- 유선 매체, 무선 매체
유선 매체
- 꼬임선(UTP, STP)
- 동축 케이블
- 광섬유
무선 매체

- 위성 통신
- 라디오파
- 적외선
꼬임선 케이블
- 저렴
- 설치 용이
- UTP, STP

- CATV 사용
광섬유

- 저렴, 단거리, 저속, 접속 쉬움
단일모드
- 고가, 장거리, 고속, 접속 어려움
광섬유 장점

- 전자기 간섭 적음
- 장거리 전송
광섬유 단점
- 설치 비용 높음
- 접속 어려움
시험 단골
- UTP/STP
- 다중모드 vs 단일모드
- 광섬유 장단점

4강 데이터 통신의 전송 기술 1

데이터 교환 방식
- 회선 교환, 패킷 교환, 메시지 교환
회선 교환(Circuit Switching)
- 연결 지향형
- 연결 설정 → 데이터 전송 → 연결 해제
- 대용량 실시간 전송 적합
패킷 교환(Packet Switching)
- 데이터를 패킷 단위로 분할
- Store-and-Forward 방식
- 헤더 오버헤드 존재
- 인터넷에서 사용
가상회선 방식
- 연결 지향형
- 회선 교환과 유사
데이터그램 방식
- 비연결형
- 패킷마다 독립 경로
메시지 교환(Message Switching)
- 메시지를 분할하지 않음
- 저장 후 전송
- 전송 지연 큼
- 실시간 서비스 부적합
교환 방식 비교
- 회선 교환 → 연결형, 실시간, 대용량
- 메시지 교환 → 비연결형, 대용량, 지연 큼
- 가상회선 → 연결형 패킷 교환
- 데이터그램 → 비연결형 패킷 교환
다중화(Multiplexing)
- 하나의 회선을 여러 사용자가 공유
시분할 다중화(TDM)
- 시간 슬롯 분할
- 디지털 방식
주파수 분할 다중화(FDM)
- 주파수 대역 분할
- 라디오, 방송
파장 분할 다중화(WDM)
- 광통신 사용
- 빛의 파장 분할
동기화
- 송수신 시점 일치
비트 동기화
- 동일 클록 사용
- 동기 신호 사용
시험 단골
- 회선 교환 vs 패킷 교환
- 가상회선 vs 데이터그램
- TDM, FDM, WDM

8강 TCP/IP : 네트워크 계층

IP
- 비연결형 서비스
- Best Effort 서비스
- IP 데이터그램 사용
IP 단편화(Fragmentation)
- MTU보다 큰 데이터 분할
MTU
- 한 번에 전송 가능한 최대 데이터 크기
ARP(Address Resolution Protocol)
- IP 주소 → MAC 주소
RARP(Reverse ARP)
- MAC 주소 → IP 주소
ICMP
- 오류 보고 및 진단
대표 예
- Ping
- Destination Unreachable
IGMP
- 멀티캐스트 그룹 관리
- 클래스 D 주소 사용
BOOTP
- 부팅 시 IP 주소 제공
DHCP
- IP 주소 자동 할당
- 임대(Lease) 방식
DNS(Domain Name System)
- 도메인 이름 ↔ IP 주소 변환
시험 단골
- ARP: IP → MAC
- RARP: MAC → IP
- DHCP: IP 자동 할당
- DNS: 이름 → IP
- ICMP: Ping
- IGMP: 멀티캐스트
- MTU, 단편화

- DHCP
- SNMP
- 실시간 스트리밍

- 연결 지향형
- 신뢰성 제공
- 흐름 제어
- 오류 제어
- 순서 보장
3-Way Handshake
- SYN
- SYN+ACK
- ACK
4-Way Handshake
- FIN
- ACK
- FIN
- ACK
시험 단골
- TCP → 연결형, 신뢰성

- TCP 3-Way Handshake
- TCP 4-Way Handshake
