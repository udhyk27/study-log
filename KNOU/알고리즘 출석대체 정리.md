# 알고리즘/자료구조 벼락치기 최종 통합 정리

## 1. 시간복잡도 순서 (매우 중요)

O(1)
< O(log n)
< O(n)
< O(n log n)
< O(n²)
< O(n³)
< O(2ⁿ)
< O(n!)

---

## 2. 점근 표기법

### Big-O : 상한 (최대 증가율)
f(n) ≤ c·g(n)

뜻:
"이 정도 이하"

예:
n² + 5n + 3 → O(n²)

그래프:
f(n)이 g(n)의 상한선을 넘지 않음

---

### Big-Omega Ω : 하한 (최소 증가율)
f(n) ≥ c·g(n)

뜻:
"최소 이 정도 이상"

그래프:
f(n)이 g(n) 아래로 안 내려감

---

### Big-Theta Θ : 정확한 증가율
c1g(n) ≤ f(n) ≤ c2g(n)

뜻:
"딱 이 정도 증가"

그래프:
f(n)이 두 곡선 사이에 존재

---

### Big-O 구하는 법

규칙:
1. 최고차항만 남김
2. 낮은 차수 제거
3. 계수 제거
4. 상수 제거

예:
n² + 5n + 3 → O(n²)
5n³ + 100n → O(n³)
5n log n + n → O(n log n)
300 → O(1)

---

## 3. 점화식 암기 (시험 단골)

T(n) = T(n/2) + 1
→ 이진 탐색
→ O(log n)

T(n) = 2T(n/2) + n
→ 합병 정렬
→ O(n log n)

T(n) = 2T(n/2) + n²
→ O(n²)

T(n) = T(n-1) + 1
→ O(n)

T(n) = T(n-1) + n
→ O(n²)

---

## 4. 탐색 알고리즘

### 순차 탐색 (Linear Search)
처음부터 끝까지 탐색

점화식:
T(n) = T(n-1) + 1

시간복잡도:
O(n)

---

### 이진 탐색 (Binary Search)
정렬된 배열에서 탐색
절반씩 제거

점화식:
T(n) = T(n/2) + 1

시간복잡도:
O(log n)

조건:
정렬되어 있어야 함

---

### 해시 탐색 (Hash Search)
해시 테이블 사용

평균:
O(1)

최악:
O(n)

특징:
충돌(collision) 가능

---

## 5. 정렬 알고리즘

### 버블 정렬 (Bubble Sort)
인접 원소 비교 후 교환

시간복잡도:
O(n²)

특징:
안정 정렬
제자리 정렬

---

### 선택 정렬 (Selection Sort)
최솟값 선택 후 앞으로 이동

시간복잡도:
O(n²)

특징:
불안정 정렬
제자리 정렬

---

### 삽입 정렬 (Insertion Sort)
정렬된 위치에 삽입

최악:
O(n²)

최선:
O(n)

특징:
안정 정렬
제자리 정렬

---

### 합병 정렬 (Merge Sort)
반으로 나눔 → 정렬 → 합침

점화식:
T(n) = 2T(n/2) + n

시간복잡도:
O(n log n)

특징:
안정 정렬
추가 메모리 필요
비제자리 정렬

---

### 퀵 정렬 (Quick Sort)
pivot 기준 분할

평균:
O(n log n)

최악:
O(n²)

특징:
불안정 정렬
제자리 정렬

---

### 힙 정렬 (Heap Sort)
힙 사용

시간복잡도:
O(n log n)

최악도:
O(n log n)

특징:
불안정 정렬
제자리 정렬

오름차순:
최대 힙 사용

내림차순:
최소 힙 사용

---

### 셸 정렬 (Shell Sort)
삽입 정렬 개선판

특징:
멀리 떨어진 원소 먼저 정렬

---

## 6. 데이터 분포 기반 정렬

### 계수 정렬 (Counting Sort)

원리:
숫자 개수 세기

시간복잡도:
O(n + k)

k = 숫자 범위

특징:
비교 기반 아님
안정 정렬
범위 제한 필요

---

### 기수 정렬 (Radix Sort)

원리:
자리수별 정렬

시간복잡도:
O(d(n+k))

d = 자리수 개수

특징:
비교 기반 아님
안정 정렬

---

### 버킷 정렬 (Bucket Sort)

원리:
범위별 bucket 분배 후 정렬

평균:
O(n)

최악:
O(n²)

특징:
데이터 분포 균등 시 빠름

---

## 16. Java 선언 / 생성 / 초기화

선언만:
int x;

선언 + 초기화:
int x = 10;

선언 후 초기화:
int x;
x = 10;

객체 선언:
Member m;

객체 생성:
m = new Member();

선언 + 생성:
Member m = new Member();

지역변수:
초기화 안 하면 사용 불가

멤버변수(필드):
자동 초기화

int → 0
double → 0.0
boolean → false
String → null

## 시험 직전 핵심 암기

Big-O = 상한
Ω = 하한
Θ = 상한+하한

Big-O = 최고차항만 남김

이진탐색 → O(log n)
합병/힙 → O(n log n)
버블/선택/삽입 → O(n²)
퀵 평균 O(n log n), 최악 O(n²)

T(n)=2T(n/2)+n
→ 합병정렬

DFS = Stack
BFS = Queue

Heap = 완전이진트리
BST = 왼 < 부모 < 오른

오름차순 힙정렬 → 최대 힙

해시 탐색 평균 → O(1)

비교 기반 정렬 최소 하한
→ O(n log n)

계수정렬 → O(n+k)
기수정렬 → 자리수별 정렬
버킷정렬 → 분포 기반 정렬
