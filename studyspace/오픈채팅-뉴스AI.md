# 쇼츠 추리소설 콘텐츠 제작 가이드 (30~40초)

## 1. 영상 구조 (핵심 포맷)

- 0~3초: 강한 훅 (사건 발생)
- 3~20초: 사건 설명 + 단서 2~3개
- 20~30초: 추리 유도 질문
- 30~40초: 반전 힌트 + 책 유도

→ 이 구조 그대로 반복 제작

## 3. 제작 흐름 (실무 프로세스)

1. ChatGPT로 대본 생성  
2. 이미지 AI로 장면 4~6개 생성  
3. TTS로 음성 생성  
4. CapCut에서 영상 제작 (이미지 + 음성 + 자막)  
5. 9:16 비율로 export 후 업로드

## 4. 대본 템플릿

[HOOK]  
한 남자가 자신의 집에서 살해됐다.  
문은 안에서 잠겨 있었고, 창문도 닫혀 있었다.  

[사건]  
경찰이 발견한 건 세 가지.  
깨진 시계, 젖은 바닥, 그리고 사라진 열쇠.  

[추리 유도]  
이 상황에서 범인은 어떻게 들어왔을까?  

[힌트]  
시계는 3시에서 멈춰 있었지만,  
이웃은 4시에 비명을 들었다.  

[마무리]  
이 트릭, 당신은 풀 수 있나?  
(이 이야기는 실제 소설을 기반으로 합니다)

## 5. ChatGPT 프롬프트

30~40초 분량의 쇼츠용 추리소설 줄거리 작성해줘.

조건:
- 초반 3초 강한 훅
- 사건 + 단서 3개 포함
- 시청자가 추리할 수 있게 질문 포함
- 정답은 말하지 말고 힌트만 제공
- 마지막에 책을 읽고 싶게 만드는 문장 추가
- 짧고 임팩트 있는 문장
