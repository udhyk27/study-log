[ 컨셉 ]

"내가 이 골목에서 들었던 그 노래, 다음 사람도 느낄 수 있도록."

장소에 음악을 핀으로 남기면, 그 자리를 지나가는 누군가가 발견하는
위치 기반 음악 방명록 앱.

카페, 공원, 골목길 — 내가 있었던 자리에 Spotify 트랙을 꽂아두면
다른 사용자가 지도에서 그 음악을 발견하고 즉시 재생할 수 있다.
인스타 스토리로 공유하면 자연스러운 바이럴로 이어진다.


[ 사용자 흐름 ]

  1. 소셜 로그인       Instagram / TikTok OAuth
  2. 위치 확인         GPS로 현재 좌표 수집
  3. 음악 검색         Spotify 트랙 검색 + 30초 미리듣기
  4. 메모 작성         한 줄 감정 + 익명/닉네임 선택
  5. 핀 드롭           지도에 음악 핀 생성
  6. 발견              주변 반경 내 음악 핀 탐색
  7. 공유              인스타 스토리 / TikTok 바이럴


[ 경쟁 앱 차별점 ]

  - 음악 = 위치의 감정   단순 핀이 아닌 그 공간의 기억을 음악으로 기록
  - 발견의 즐거움        근처 핀 진동 알림 → 우연한 타인의 이야기
  - 소셜 바이럴 구조     앨범아트 + 지도 + QR 합성 카드 자동 생성

[ 기능 목록 ]


  ── 핵심 기능 (MVP) ──

  1. 소셜 로그인
     Instagram Graph API + TikTok Login Kit
     OAuth 2.0 방식, 별도 회원가입 없이 바로 시작

  2. 음악 핀 남기기
     현재 GPS 좌표 + 트랙 ID + 한 줄 메모 저장
     익명 또는 닉네임 선택 가능

  3. 주변 음악 탐색 지도
     반경 500m ~ 5km 내 핀 클러스터링 표시
     핀 탭 → 30초 미리듣기 재생

  4. 음악 검색 연동
     Spotify Search API
     앨범아트 · 아티스트 · 미리듣기 URL 포함

  5. 인스타 스토리 공유
     앨범아트 + 지도 썸네일 + 앱 QR 합성 이미지 자동 생성
     Instagram Content Publishing API로 스토리 공유

  6. TikTok 공유
     TikTok Share API로 음악 + 지도 카드 공유


  ── 추가 제안 기능 ──

  7.  근처 핀 알림 (푸시)
      백그라운드 위치 감지 → 반경 내 새 핀 등록 시 FCM 알림
      재방문 유도 핵심 기능

  8.  핀 댓글
      익명 · 닉네임으로 짧은 반응 남기기
      "이 곡 저도 여기서 들었어요" 같은 공감 문화 형성

  9.  핀 좋아요 / 수집
      마음에 드는 핀을 컬렉션에 저장
      "내가 수집한 음악 지도" 모아보기 페이지

  10. 도시별 인기 핀 랭킹
      서울 · 부산 · 제주 등 도시 단위 주간 랭킹
      콘텐츠 소비 허브 역할

  11. 무드 태그
      #새벽감성 #비오는날 #설레는 등 해시태그 선택
      태그 기반 필터 탐색 가능

  12. 24H 임시 핀
      24시간 후 자동 삭제되는 휘발성 핀
      인스타 스토리처럼 더 솔직한 감정 표현 유도

  13. 커스텀 핀 스킨  [PREMIUM]
      시즌 한정 · 아티스트 콜라보 핀 스킨
      유료 구매 또는 팔로워 수 달성 보상

  14. 나의 음악 발자국  [PREMIUM]
      내가 남긴 핀 히스토리 지도
      연말 "올해 내 음악 발자국" 리포트 공유
