# 상태 표현과 행동 표현 정리

## 기본 개념

상태 표현  
명사 형용사 뒤에 です 사용  
현재 상태 느낌  

행동 표현  
동사 뒤에 ます 사용  
행동 동작 느낌

## 비교 예시

私は学生です  

저는 공부합니다  
私は勉強します

この映画は面白いです  

지금 영화를 보고 있습니다  
今映画を見ています
