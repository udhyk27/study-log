** TAB  → 접근 가능한 객체
** TABS (= USER_TABLES) → 내가 소유
** USER_TAB_COLUMNS → 내 테이블의 컬럼

---------------------------------------
# 테이블 속성
---------------------------------------
[테이블]
USER_TABLES       → 내 테이블
ALL_TABLES        → 접근 가능한 테이블
DBA_TABLES        → 전체 테이블

[컬럼]
USER_TAB_COLUMNS  → 내 테이블 컬럼
ALL_TAB_COLUMNS   → 접근 가능한 컬럼

[시퀀스]
USER_SEQUENCES    → 내 시퀀스

[뷰]
USER_VIEWS        → 내 뷰

[인덱스]
USER_INDEXES      → 내 인덱스

[제약조건]
USER_CONSTRAINTS  → PK, FK, UNIQUE 등
USER_CONS_COLUMNS → 제약조건 컬럼

[객체]
TAB               → 접근 가능한 객체 목록

---------------------------------------
# 날짜
---------------------------------------
SYSDATE        : 현재 날짜
SYSTIMESTAMP   : 현재 날짜/시간

YYYYMMDDHH24MISS

ADD_MONTHS()   : 월 계산
LAST_DAY()     : 월 마지막날
TRUNC()        : 시간 제거/절삭

** 조건
오늘
TRUNC(DATE) = TRUNC(SYSDATE)

기간
BETWEEN A AND B

최근 7일
DATE >= SYSDATE - 7

이번달
DATE >= TRUNC(SYSDATE,'MM')
AND DATE < ADD_MONTHS(TRUNC(SYSDATE,'MM'),1)

---------------------------------------
# 실무 함수
---------------------------------------
** NVL2(SAL, '있음', '없음')

** DECODE()     : 조건 분기
```
SELECT DECODE(DEPTNO,
			  10, '총무',
              20, '개발',
              30, '영업',
              '기타')
FROM EMP;
```

** CASE WHEN    : 조건 분기
CASE
    WHEN SAL >= 5000 THEN 'A'
    WHEN SAL >= 3000 THEN 'B'
    ELSE 'C'
END

** MERGE
MERGE INTO 대상테이블 A
USING 원본테이블 B
ON (조건)

WHEN MATCHED THEN
    UPDATE SET ...

WHEN NOT MATCHED THEN
    INSERT (...)
    VALUES (...);
---------------------------------------
# ALTER
---------------------------------------
** 컬럼 추가
ALTER TABLE 테이블명
ADD 컬럼명 컬럼타입;

** 컬럼 수정
ALTER TABLE 테이블명
MODIFY 컬럼명 컬럼타입;

** 컬럼명 변경
ALTER TABLE 테이블명
RENAME COLUMN 기존컬럼명
TO 새컬럼명;

** 컬럼 삭제
ALTER TABLE 테이블명
DROP COLUMN 컬럼명;

** 제약조건 추가
ALTER TABLE 테이블명
ADD CONSTRAINT 제약조건명
PRIMARY KEY(컬럼명);

** 제약조건 삭제
ALTER TABLE 테이블명
DROP CONSTRAINT 제약조건명;

** NOT NULL 추가
ALTER TABLE 테이블명
MODIFY 컬럼명 NOT NULL;

** DEFAULT 값 설정
ALTER TABLE 테이블명
MODIFY 컬럼명 DEFAULT 값;

** 테이블명 변경
RENAME 기존테이블명
TO 새테이블명;
